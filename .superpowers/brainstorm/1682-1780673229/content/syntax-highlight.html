深入理解 JavaScript 闭包

什么是闭包？

闭包是指一个函数能够记住并访问它的词法作用域，即使这个函数在它的词法作用域之外执行。

function outer() {
  const name = "Closure"
  return function inner() {
    console.log(name)
  }
}

const fn = outer()
fn() // "Closure"

场景说明
数据私有模拟 private
函数工厂动态创建函数
柯里化部分应用参数
