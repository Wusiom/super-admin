Vue 3 Markdown 编辑器库对比

三款现成的分屏编辑器，无需从零造轮子

A

md-editor-v3

最流行，CodeMirror 5 + Markdown 渲染。内置分屏预览、同步滚动、暗色主题、全屏、TOC 目录。

Pros

  • 开箱即用，API 简洁
  • 暗金主题可自定义
  • 10.6k GitHub stars
  • 中文文档齐全

Cons

  • CodeMirror 5 (非 v6)
  • 包体积偏大

npm: @kangc/v-md-editor (next 版本支持 Vue 3)

B

md-editor-v3 (新一代)

CodeMirror 6 底层，专为 Vue 3 重写。三种模式（edit/split/preview）、插件系统、拖拽调整窗格。

Pros

  • CodeMirror 6 — 现代化架构
  • Vue 3 原生支持
  • previewOnly 模式即纯阅读
  • TypeScript 类型完善

Cons

  • 较新，社区略小
  • 主题定制需要 CSS 变量

npm: md-editor-v3

C

vue-markdown-editor

模块化设计，可选 textarea 或 CodeMirror 引擎。轻量灵活，按需引入。

Pros

  • 轻量基础版 (textarea)
  • 可升级到 CodeMirror
  • 模块化、按需引入

Cons

  • 基础版功能少
  • 配置稍复杂

npm: @kangc/v-md-editor
