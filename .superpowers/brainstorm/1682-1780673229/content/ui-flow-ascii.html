🔄 状态流转


  ┌─────────────┐    点击行     ┌─────────────────┐
  │             │ ────────────▶ │                  │
  │  知识列表    │              │  Dialog (预览)    │
  │             │ ◀──────────── │  mode="preview"  │
  └─────────────┘    关闭弹窗    │                  │
                                └───────┬─┬───────┘
                                        │ │
                             [编辑] ────┘ │ [全屏] ────┐
                                │         │             │
                                ▼         │             ▼
                    ┌──────────────┐     │   ┌──────────────────┐
                    │ Dialog (编辑) │     │   │  MarkdownEditPage │
                    │ mode="split" │     │   │  /edit/:id        │
                    │              │     │   │  mode="split"     │
                    └──────┬───────┘     │   └────────┬─────────┘
                           │             │            │
                    [预览] ─┘             │     ← 返回列表
                           │             │
                    Ctrl+S │             │
                           ▼             │
                    ┌──────────────┐     │
                    │  PUT /items  │     │
                    │  :id (保存)   │ ◀───┘  (全屏页也能保存)
                    └──────────────┘

🧩 MarkdownEditor.vue Props/Events

  Props:                    Events:                   Slots:
  ─────                     ──────                    ─────
  modelValue: string        update:modelValue         toolbar-left
  mode: 'preview'           save (Ctrl+S 触发)        toolbar-right
         | 'edit'
         | 'split'
  height: string (默认 60vh)
  readonly: boolean (纯阅读时 true)
