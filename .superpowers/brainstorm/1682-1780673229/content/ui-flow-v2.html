🧩 MarkdownEditor.vue Props（简化后）

  Props:                    Events:
  ─────                     ──────
  modelValue: string        update:modelValue    (输入即更新 v-model)
  height?: string           save                 (Ctrl+S → emit)
  readonly?: boolean        (全屏页不需要)

  默认行为：
  - 始终分屏 (split)
  - Ctrl+S 自动监听 → emit('save')
  - 编辑区 CodeMirror 6 + 预览区 Markdown 渲染 + highlight.js
