深入理解 JavaScript 闭包

什么是闭包？

闭包是指一个函数能够记住并访问它的词法作用域，即使该函数在它的词法作用域之外执行。

核心示例

function createCounter() {
  let count = 0;
  return function() {
    count++;
    return count;
  };
}

const counter = createCounter();
console.log(counter()); // 1
console.log(counter()); // 2

实际应用

  • 模块模式：封装私有变量
  • 事件处理器：保持回调上下文
  • 柯里化：部分应用函数参数
闭包是 JavaScript 最强大的特性之一，理解它对于掌握异步编程至关重要。

深入理解 JavaScript 闭包

什么是闭包？

闭包（Closure）是指一个函数能够记住并访问它的词法作用域，即使该函数在它的词法作用域之外被执行。这是 JavaScript 最核心的概念之一。

词法作用域

JavaScript 采用词法作用域（静态作用域），这意味着函数的作用域在函数定义时就已经确定，而不是在函数调用时。

经典示例

function outer() {
  const message = 'Hello';

  function inner() {
    console.log(message);
  }

  return inner;
}

const fn = outer();
fn(); // 'Hello' — 闭包让 inner 记住了 message

常见用途

  1. 数据私有化（模块模式）
  2. 回调函数与事件处理
  3. 函数工厂与柯里化
  4. 循环中的异步问题

注意事项

闭包会持有对外部变量的引用，可能导致内存泄漏。在不需要时应及时解除引用。
